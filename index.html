Sistema de Gestão de Estoque da Konecta Brazil

Julho 2022 - Setembro 2022
Link do projeto

O Sistema de gestão do estoque tinha por objetivo reduzir a quantidade de planilhas presentes na Konecta Brazil, o objetivo do sistema era ajudar a parte de suporte técnico da empresa e gerenciar todo o estoque de equipamentos através do sistema, de forma rápida e intuitiva.
O sistema foi desenvolvido utilizando os Frameworks ASP.NET Core e o Bootstrap, as linguagens C#, HTML, CSS e o JavaScript e o banco de dados SQL Server

Mangá Store

Agosto 2022 - Dezembro 2022
Link do Projeto

A Mangá Store foi um projeto desenvolvido durante a realização do 4º Semestre do curso de Análise e Desenvolvimento de Sistemas. Ele consiste num site de e-commerce de mangás, junto com ele foi desenvolvido um plano de testes.
Foram utilizadas boas práticas de desenvolvimento e testes de software, fazendo com que o sistema fosse desenvolvido principalmente nas aulas de Projeto Integrador 4 e Testes de Software.
O sistema foi desenvolvido utilizando os Frameworks ASP.NET Core e o Bootstrap, as linguagens C#, HTML, CSS e o JavaScript e o banco de dados SQL Server. Os testes foram realizados utilizando ferramentas especializadas como o Apache JMeter, o Lighthouse e o Roslyn, que é a ferramenta de manutenibilidade nativa do .NET.
